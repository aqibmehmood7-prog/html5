Course Description

This course covers HTML5, CSS3, web servers, RESTful web services and client-side scripting with JavaScript. It emphasizes practical exercises and validated, accessible markup.

What you will learn

  • HTML5 semantic structure and accessibility basics
  • Styling with CSS3 and responsive design
  • Introduction to REST APIs and JSON
  • Basic server configuration and deployment
